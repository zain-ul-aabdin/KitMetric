About KitMetric

KitMetric is a lightweight toolkit for image utilities and developer workflows. The shared header and footer are loaded from reusable partials so every page stays consistent.

Shared LayoutHeader and footer are now applied site-wide through reusable HTML partials.
Static Site FriendlyUses a small client-side loader to include shared markup on plain HTML pages.
Reusable PatternAdd more pages easily with the same partials and stylesheet.

How it works

This page loads the shared header and footer from the `partials` folder using `partials/includes.js`.
